## Problem Statement
```
You are given an integer array nums and an integer target.

Return the number of subarrays of nums in which target is the majority element.

The majority element of a subarray is the element that appears strictly more than half of the times in that subarray.
```

### Constraints:
```
1 <= nums.length <= 1000
1 <= nums[i] <= 10^9
1 <= target <= 10^9
```


## Approach-1: By visiting each & every sub-array

---

### Code:
```java
class Solution {
    public int countMajoritySubarrays(int[] nums, int target) {
        int n = nums.length;
        int ans = 0;
        for (int i = 0; i < n; i++) {
            int count = 0;
            for(int j=i;j<n;j++){
                if(nums[j]==target){
                    count += 1;
                }
                if(count > (j-i+1)/2){
                    ans+=1;
                }
            }
        }
        return ans;
    }
}
```

### Complexity
```
Time : O(n^2) As we are using two-loops.
```
