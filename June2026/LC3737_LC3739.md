## Problem Statement
```
You are given an integer array nums and an integer target.

Return the number of subarrays of nums in which target is the majority element.

The majority element of a subarray is the element that appears strictly more than half of the times in that subarray.
```

### Constraints:
```
1 <= nums.length <= 1000
1 <= nums[i] <= 10^9
1 <= target <= 10^9
```


## Approach-1: By visiting each & every sub-array
- Initialize a variable 'ans=0' to keep track of the number of sub-arrays who are satisfying given condition.
- Run a loop from i=0 to n-1 which will represent the starting point of a subarray.
- Initialize a variable 'count=0' to keep track of number of target element in the current sub-array.
- Run another nested loop from j=i to j=n-1 which will represent ending point of a subarray.
  - Whenever we get value==target will increase count by 1.
  - if count > (j-i+1)/2 will increase 'ans' by 1.
- Return 'ans'.

---

### Code:
```java
class Solution {
    public int countMajoritySubarrays(int[] nums, int target) {
        int n = nums.length;
        int ans = 0;
        for (int i = 0; i < n; i++) {
            int count = 0;
            for(int j=i;j<n;j++){
                if(nums[j]==target){
                    count += 1;
                }
                if(count > (j-i+1)/2){
                    ans+=1;
                }
            }
        }
        return ans;
    }
}
```

### Complexity
```
Time : O(n^2) As we are using two nested-loops.
```

## Approach-2: Using +1 , -1 logic

- Same as previous one here just add +1 if element==target in current sub-array window otherwise, add -1.
- And whenever count > 0 (means we have more than size/2 target elements in the current sub-array) increase subarray count by 1.


---

### Code:
```java
class Solution {
    public int countMajoritySubarrays(int[] nums, int target) {
        int n = nums.length;
        int ans = 0;
        for (int i = 0; i < n; i++) {
            int count = 0;
            for(int j=i;j<n;j++){
                if(nums[j]==target){
                    count += 1;
                }else{
                    count-=1;
                }
                if(count > 0){
                    ans+=1;
                }
            }
        }
        return ans;
    }
}
```

### Complexity
```
Time : O(n^2) As we are using teo nested-loops.
```

## Approach-3: Using prefixSum logic
- Create a transformed array where: 
  - target is represented as +1. 
  - Every other element is represented as -1.
- Compute the prefix sum array prefix[] of the transformed values.
- Initialize a variable ans = 0 to store the number of valid subarrays.
- Run a loop from i = 0 to n - 1 to represent the starting index of a subarray.
- Run another nested loop from j = i to n - 1 to represent the ending index of a subarray.
  - Calculate the transformed sum of the subarray: subSum = prefix[j + 1] - prefix[i]. 
  - If subSum > 0, it means the number of occurrences of target is greater than the number of non-target elements in that subarray. Increment ans by 1.
- Return ans.

---

### Code:
```java
class Solution {
    public int countMajoritySubarrays(int[] nums, int target) {
        int n = nums.length;
        int ans = 0;

        int[] cumSum = new int[n];
        int sum = 0;
        for (int i = 0; i < n; i++) {
            if (nums[i] == target) {
                sum += 1;
            } else {
                sum -= 1;
            }
            cumSum[i] = sum;
        }

        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                int subSum;
                if (i == 0) {
                    subSum = cumSum[j];
                } else {
                    subSum = cumSum[j] - cumSum[i - 1];
                }
                if (subSum > 0) {
                    ans++;
                }
            }
        }
        return ans;
    }
}
```

### Complexity
```
Time : O(n^2) As we are using two nested loops.
Space : O(n)
```

## Approach-4: Using HashMap
- Convert the problem into a prefix sum problem: 
  - target → +1 
  - non-target → -1
- A subarray is valid if its transformed sum is greater than 0.
- Maintain:
  - sum = current transformed prefix sum. 
  - mp = frequency of each prefix sum seen so far. 
  - count = number of previous prefix sums smaller than the current prefix sum (i.e., valid subarrays ending at the current index). 
  - ans = total number of valid subarrays.
- Initialize: sum = 0, count = 0, ans = 0, mp[0] = 1 
- Traverse the array:
  - If the current element is target:
    - Prefix sum increases by 1. 
    - Prefix sums equal to the old sum become newly smaller.
  - Otherwise:
    - Prefix sum decreases by 1. 
    - Prefix sums equal to the new sum are no longer smaller.
  - Add count to ans.
  - Store the updated prefix sum in mp.
- Return ans.

---
### Code:
```java
class Solution {
  public long countMajoritySubarrays(int[] nums, int target) {
    int n = nums.length;
    Map<Integer,Integer> mp = new HashMap<>();

    int sum = 0;
    mp.put(0,1);
    int count = 0;
    long ans = 0;

    for(int i=0;i<n;i++){
      if(nums[i]==target){
        count += mp.getOrDefault(sum,0);
        sum+=1;
      }
      else{
        sum -= 1;
        count -= mp.getOrDefault(sum,0);
      }
      ans += 1L * count;
      mp.put(sum,mp.getOrDefault(sum,0)+1);
    }
    return ans;
  }
}
```

### Complexity
```
Time : O(n)
Space : O(n) As range of sum can be -n to n.
```
