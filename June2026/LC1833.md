# Max Ice Cream Bars Problem

## Problem Statement
You are given an array `costs` where `costs[i]` is the cost of the ith ice cream bar and an integer `coins`.  
You can buy at most one ice cream bar of each type.  
Return the maximum number of ice cream bars you can buy.

---
# Approaches

---

### Brute-force: Using sorting
- Sort the array (costs) in ascending order. 
- Initialize a variable "count" to keep track of number of ice-bars. 
- Do a traversal((left to right)) 
- During traversal whenever we get cost[i]<=coins will decrease the coins=coins-cost[i] & increase count by 1. 
- At the end will return the count.



### Code:
```java
import java.util.Arrays;

class Solution {
    public int maxIceCream(int[] costs, int coins) {
        Arrays.sort(costs);
        int count = 0;

        for (int e : costs) {
            if (e <= coins) {
                coins -= e;
                count += 1;
            }
        }
        return count;
    }
}
```

### Optimal: Using count sort
- First find the max element in the given array.
- After that create an array of (maxEle+1) size & initialize it with zero.
- Traverse the costs array and store the frequency of each cost in newly created array.
- Initialize count to keep track of number of ice-cream bars.
- Do a traversal(left to right) on created array For each cost value, while its frequency is greater than zero and the cost is less than or equal to the remaining coins, subtract the cost from coins, decrease its frequency, and increase the count of ice creams purchased. If coins become zero, stop the process.
- At the end will simply return the count.



### Code:
```java
class Solution {
    public int maxIceCream(int[] costs, int coins) {
        int max = Integer.MIN_VALUE;
        for(int e:costs){
            max=Math.max(max,e);
        }

        int[] temp = new int[max+1];
        for(int e: costs){
            temp[e]+=1;
        }

        int count=0;
        for(int i=0;i<temp.length;i++){
            if(coins==0) break;
            int freq = temp[i];
            while(freq>0 && i<=coins){
                coins-=i;
                freq-=1;
                count+=1;
            }
        }
        return count;
    }
}
```
