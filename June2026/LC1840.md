## Problem Statement
You want to build n new buildings in a city. The new buildings will be built in a line and are labeled from 1 to n.

However, there are city restrictions on the heights of the new buildings:

The height of each building must be a non-negative integer.
The height of the first building must be 0.
The height difference between any two adjacent buildings cannot exceed 1.
Additionally, there are city restrictions on the maximum height of specific buildings. These restrictions are given as a 2D integer array restrictions where restrictions[i] = [idi, maxHeighti] indicates that building idi must have a height less than or equal to maxHeighti.

It is guaranteed that each building will appear at most once in restrictions, and building 1 will not be in restrictions.

Return the maximum possible height of the tallest building.

### Constraints:
- 2 <= n <= 10^9
- 0 <= restrictions.length <= min(n - 1, 10^5)
- 2 <= idi <= n
- idi is unique.
- 0 <= maxHeighti <= 10^9

---
# Approaches

---

### Brute-force: 
- Create an array of 'height' (n+1) size which will represents buildings.
- Traverse restrictions array & assign height to respective building in 'height'.
- Traverse 'height' from left to right starting at position to 2. For every building assign it's correct height according to left neighbour. To find correct height of a building according to left neighbour use formulae ``` Math.min(currentBuildingHeight, prevBuildingHeight + (currentBuildingPosition - prevBuildingPosition)) ```
- Traverse 'height' from right to left starting at position n-1. For every building assign it's correct height according to right neighbour. To find correct height of a building according to right neighbour use formulae ``` Math.min(currentBuildingHeight, nextBuildingHeight + (nextBuildingPosition - currentBuildingPosition)) ```
- Traverse 'height' & take max which will be our answer.
- We will get TLE on submitting as here number of buildings can be 10^9.

---

### Optimal:
- In this we will traverse the restrictions array as it's size can be max 10^5.
- Add (1,0) and (n,n-1)
- Sort the restrictions array.
- Traverse 'restrictions' from left to right starting at position to 2. For every building assign it's correct height according to left neighbour. To find correct height of a building according to left neighbour use formulae ``` Math.min(currentBuildingHeight, prevBuildingHeight + (currentBuildingPosition - prevBuildingPosition)) ```
- Traverse 'restrictions' from right to left starting at position n-1. For every building assign it's correct height according to right neighbour. To find correct height of a building according to right neighbour use formulae ``` Math.min(currentBuildingHeight, nextBuildingHeight + (nextBuildingPosition - currentBuildingPosition)) ```
- Initialize one var maxHeight to keep track of max height of a building.
- Traverse 'restrictions' from left to right starting at position to 2. For every building find maxHeight of building in between currentBuilding & previousBuilding using formulae ```max(prevBuildingHeight,currentBuildingHeight) + ( (currentBuildingPosition - prevBuildingPosition) - Math.abs(prevBuildingHeight - currentBuildingHeight) )/2 ```



### Code:
```java
class Solution {
    public int maxBuilding(int n, int[][] restrictions) {
        List<int[]> r = new ArrayList<>();
        for (int[] res : restrictions) {
            r.add(new int[] { res[0], res[1] });
        }
        r.add(new int[] { 1, 0 });
        r.add(new int[] { n, n - 1 });
        r.sort((a, b) -> Integer.compare(a[0], b[0]));

        int m = r.size();
        for (int i = 1; i < m; i++) {
            int gap = r.get(i)[0] - r.get(i - 1)[0];
            int h1 = r.get(i - 1)[1];
            int h2 = r.get(i)[1];
            r.get(i)[1] = Math.min(h2, h1 + gap);
        }

        for (int i = m - 2; i >= 0; i--) {
            int gap = r.get(i + 1)[0] - r.get(i)[0];
            int h1 = r.get(i + 1)[1];
            int h2 = r.get(i)[1];
            r.get(i)[1] = Math.min(h2, h1 + gap);
        }

        int ans = 0;
        for (int i = 1; i < m; i++) {
            int gap = r.get(i)[0] - r.get(i - 1)[0];
            int h1 = r.get(i - 1)[1];
            int h2 = r.get(i)[1];
            int best = Math.max(h1, h2) + (gap - Math.abs(h1 - h2)) / 2;
            ans = Math.max(ans, best);
        }
        return ans;
    }
}
```
